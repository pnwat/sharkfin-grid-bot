# MSTR/BTC裁定とシャークフィンの相性分析

## 現状

### シャークフィン
- 取引ペア: BTC/USDT
- 戦略: グリッド（買い・売り交互）
- エクスポージャー: BTCロング・ショート混在

### MSTR/BTC裁定（検討中）
- 取引ペア: MSTR/USDT + BTC/USDT
- 戦略: MSTRロング + BTCショート
- エクスポージャー: MSTRロング、BTCショート固定

---

## 問題点

### 1. BTCエクスポージャーの重複

シャークフィンがBTCポジションを持っている状態で、
MSTR裁定のBTCショートを入れると：

```
シャークフィン: BTC +0.008（ロング）
MSTR裁定: BTC -0.015（ショート）
----------------------------
ネット: BTC -0.007（ショート）
```

→ シャークフィンのヘッジが崩れる可能性

### 2. レバレッジの積み上げ

| ボット | 最大エクスポージャー |
|--------|----------------------|
| シャークフィン | $1,000 |
| MSTR裁定 | $2,000（$1,000×2） |
| **合計** | **$3,000** |

残高$1,081に対して$3,000 = **2.8倍レバレッジ**

### 3. 相関リスク

MSTR/BTC相関が崩れた場合：
- MSTRが急落（BTCは横ばい）
- シャークフィンのBTCロングも損失
- **両方で損失**の可能性

---

## 解決策

### 案1: 別口座で運用
- シャークフィン: 口座A
- MSTR裁定: 口座B

→ リスク分離可能

### 案2: シャークフィンのBTCポジションを考慮
```python
# MSTR裁定のBTCショート量を調整
mstr_position = 1000  # MSTRロング
btc_hedge = mstr_position * 0.7  # 相関70%

# シャークフィンのBTCポジションを引く
sharkfin_btc = 500  # 現在のBTCエクスポージャー
adjusted_btc_hedge = btc_hedge - sharkfin_btc

# 結果
# MSTR: +$1000
# BTC: -$200（調整後）
```

### 案3: シャークフィンを停止してMSTR裁定に集中
- MSTR裁定の週収益: $1,043
- シャークフィンの週収益: ~$35-70

→ MSTR裁定の方が収益性が高い

---

## 推奨

**現在はシャークフィンのみ運用**
- MSTR裁定は別途検討
- または、シャークフィンのBTCポジションをモニタリングしながら調整

   - シャークフィン + MSTR裁定の合計
   - $500以内に制限

2. **ストップロス連動**
   - どちらかが発動したら両方停止

3. **証拠金余力確認**
   - 最低$500は残す
