# MSTR/BTCファンディング裁定 - 深層分析

## 1. MSTR（マイクロストラテジー）とは

### 企業概要
- **事業**: ビジネスインテリジェンスソフトウェア企業
- **設立**: 1989年
- **時価総額**: 約$40B（2026年4月現在）
- **特徴**: 世界最大のBTC保有企業（214,000 BTC = $14.3B）

### BTCとの相関
```
相関係数: 0.70-0.90（期間による）
ベータ値: 2.0-3.0（BTCの2-3倍のボラティリティ）
```

**MSTR価格 = BTC価格 × 保有BTC数 × プレミアム係数**

## 2. なぜMSTRのファンディングが高いのか

### 需給ギャップ
```
MSTRロング需要（機関投資家）
    >
MSTRショート供給（GRVTトレーダー）
```

### 原因分析
1. **機関投資家の需要**: MSTR株を間接的に保有したい
2. **供給不足**: MSTRはアルトコイン扱いで流動性が低い
3. **シャート難**: MSTRを借りてショートするのが困難

### GRVTでのファンディング履歴（推定）
| 期間 | ファンディング |
|------|----------------|
| 2026年3月 | 2-5% / 8時間 |
| 2026年4月 | **4.18% / 8時間** |

## 3. リスク詳細分析

### 3.1 ファンディングレート低下リスク

**シナリオ**: MSTRショートが増えてファンディングが下がる

| ファンディング | 週収益（$1,000） |
|----------------|------------------|
| 4.18% | $1,043 |
| 2.00% | $500 |
| 1.00% | $250 |
| 0.50% | $125 |

**軽減策**: ファンディングが1%を下回ったら停止

### 3.2 MSTR/BTC相関崩壊リスク

**シナリオ**: MSTRだけが急落（BTCは横ばい）

```
MSTR -10%（BTC ±0%）
→ ポジション損失: -$100（$1,000ポジション）
→ ファンディング収益: +$125/8時間
→ 8時間で取り返し可能
```

**履歴的相関崩壊事例**:
- 2024年: MSTRがSEC調査で-15%（BTCは±0%）
- 相関は数日で回復

### 3.3 流動性リスク

**MSTR/USDT出来高**: BTCの約1/10

```
スリッページ: 0.1-0.5%
注文サイズ制限: $5,000-10,000
```

**軽減策**: ポジションサイズを$1,000以下に制限

### 3.4 GRVTリスク

**取引所リスク**:
- ハッキング
- 倒産
- 出金停止

**軽減策**: 資金の30%以上をGRVTに置かない

## 5. 最適ポジションサイズ

### ケリー基準
```
f* = (p * b - q) / b
p = 勝率 = 0.95
q = 敗率 = 0.05
b = オッズ = 4.18 / 1（リスク）

f* = (0.95 * 4.18 - 0.05) / 4.18 = 0.93
```

**ケリー基準**: 口座の93%→危険すぎる

### 安全なサイズ
```
半ケリー: 46% = $500
四分ケリー: 23% = $250
```

**推奨**: $500（口座の約50%）

## 7. シャークフィンとの併用

### 問題点
```
シャークフィン: BTCグリッド（ロング・ショート混在）
MSTR裁定: BTCショート固定

→ BTCエクスポージャーが複雑になる
→ ヘッジが崩れる可能性
```

### 解決策
1. **別口座**: シャークフィンとMSTR裁定を分ける
2. **ネット調整**: MSTR裁定のBTCショート量を調整
3. **選択**: 収益性が高い方を選ぶ

### 収益比較（$1,000口座）

| 戦略 | 週収益 | リスク | 手間 |
|------|--------|--------|------|
| シャークフィン | ~$35 | 低 | 自動 |
| **MSTR裁定** | **$1,043** | 中 | 半自動 |
